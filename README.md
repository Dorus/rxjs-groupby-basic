# rxjs-groupby-basic
Basic demo of `groupBy` for [RxJS Live](https://www.rxjs.live).

Each branch demonstrates a different operator: `mergeMap`, `switchMap`, `concatMap`, `exhaustMap`, `groupBy`, and a custom operator implementation of `groupBy`.
